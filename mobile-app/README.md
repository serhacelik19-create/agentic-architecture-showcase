# Kortex Mobile App

Flutter-based student mobile application for the Kortex education platform.

The app is built around YKS preparation workflows in the Turkish education system, including study tracking, question solving, practice flows, exam/content screens, parent access, and student progress features.

## Local Setup

```bash
flutter pub get
flutter analyze
```

The public demo version points to local backend endpoints only and does not include production secrets, Firebase configuration, or external service credentials.
